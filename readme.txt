Display a complete inventory of all WordPress content with their assigned templates, taxonomies, and other key information.

== Description ==

Content Inventory is a comprehensive tool designed to help WordPress site administrators and content managers get a complete overview of their website content.

The plugin provides a user-friendly interface in the WordPress admin area that displays:

* All pages with their assigned templates
* All custom post types with their assigned taxonomies
* Detailed information about content including URLs, IDs, titles, and statuses
* Ability to filter content by post type, taxonomy, term, status, and template
* Export functionality to CSV for further analysis

Whether you're planning a site redesign, content migration, or just need to understand your content structure better, Content Inventory gives you the insights you need.

= Features =

* Filter content by post type
* View template usage across all pages
* Filter by taxonomy and term
* Filter by content status
* Export filtered content to CSV
* See all taxonomies assigned to content

== Installation ==

1. Upload the `content-inventory` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the tool via the 'Content Inventory' menu in your WordPress admin area

== Frequently Asked Questions ==

= Does this plugin modify my content? =

No, Content Inventory is a read-only tool that only displays information about your existing content. It does not modify any content or settings.

= Can I export the audit results? =

Yes, you can export the results to a CSV file for further analysis in spreadsheet software.

= Will it work with custom post types? =

Yes, Content Inventory works with all public post types including built-in and custom types.
